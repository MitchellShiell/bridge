# `/docs`

The following pages describe the minimum standard of developer documentation required within the `/docs` folder of an Overture project:

    ```
    .
    └── docs/
        ├── 01-Overview/
        │   └── 01-Overview.md
        ├── 02-Setup.md
        ├── 03-Guides/
        │   ├── 01-Usage.md
        │   ├── 02-Development.md
        │   └── 03-Deployment.md
        └── 04-Glossary.md
    ```
